# nilhmm-paper

Preprint / manuscript for the **nilHMM** methods paper: *Simulation-calibrated
HMM ancestry calling for near-isogenic lines across sequencing modalities.*

This is a **standalone git repository** (its own history/remote). It lives inside
the `zealhmm` working tree for convenience but is **gitignored by zealhmm** — the
two are independent repos.

## Template lineage

- **Base:** the [RILAB bioRxiv template](https://github.com/RILAB/rilab_template)
  (`rilabRxiv.cls`, `genetics.bst`) — the lab's preprint styling and workflow.
- **Structure:** organized to the **Oxford Bioinformatics** layout — a structured
  abstract (Motivation / Results / Availability / Contact), Introduction /
  Materials and Methods / Results / Discussion, and OUP back-matter (Conflicts of
  interest, Funding, Data availability, Author contributions), reproduced in
  RILAB-compatible LaTeX.
- **Submission port:** at submission, port to OUP's `oup-authoring-template.cls`
  (native structured `\abstract`, `\ORCID`, `\boxedtext`). The OUP bundle is kept
  as reference in `zealhmm/agent/oup_template/` (not copied here).

## Layout

```
main.tex            # the manuscript (RILAB class + Bioinformatics structure)
rilabRxiv.cls       # RILAB document class
genetics.bst        # bibliography style
references.bib      # bibliography (seeded from the RILAB example; replace)
cornkey.png         # lab header/footer logo (used by the class)
figures/            # manuscript figures F1–F4 (generated by zealhmm)
```

## Build

```sh
latexmk -pdf main.tex        # or: pdflatex → bibtex → pdflatex ×2
```

`\todo{...}` margin notes mark what remains to be written; `\citex{...}` marks
citations to add. Figures are produced by the
[`zealhmm`](https://github.com/sawers-rellan-labs/zealhmm) analysis repo and
dropped into `figures/`.
